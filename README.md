# MaanGroup - Strategic Professional Solutions

**Live Website:** [https://maangroup.so/](https://maangroup.so/)

Maan Group is a leading consultancy and training firm in Somalia, dedicated to empowering organizations through strategic advisory, leadership development, capacity building, and applied research. We provide practical, results-driven solutions rooted in localized insights with a focus on modern, human-centric design.

## 🛠️ Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Framer Motion, Lucide-Icons

## 📦 Getting Started

```bash
git clone https://github.com/Amen252/Maan-Group.git
cd Maan-Group
npm install
npm run dev
```

Open your browser and navigate to: `http://localhost:5173`
